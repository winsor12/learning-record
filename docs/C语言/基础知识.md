# 基础知识

> 参考《C Primer Plus》

## 第一个程序

```c
#include <stdio.h>
int main(void) {
	int num;
	num = 1;
	printf("Hello world");
	printf("The num is:%d", num);
	return 0;
}
```

### 头文件

`#include <stdio.h>`

预处理指令，再程序编译前将stdio.h文件内容拷贝到当前文件内。

### main函数

程序从mian()函数开始执行。

### 代码块

由`{}`括起来的代码。

### 声明

`int num;`

在内存中分配存储空间，存储名称为”num“的int类型变量。
